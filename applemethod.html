The apple method

The Apple Method is a method for solving algebra problems. An apple is used to make a clever algebraic substitution.

Evaluate: $$\sqrt{6+\sqrt{6+\sqrt{6+\ldots}}}$$

We can solve this by using the apple method. Let $(^{^(})=\sqrt{6+\sqrt{6+\sqrt{6+\ldots}}}$. Notice that $(^{^(})$ actually appears inside itself: $(^{^(})=\sqrt{6+(^{^(})}$ Now, we can solve this by squaring both sides and factoring: $$(^{^(})^2=6+(^{^(})$$ $$\left((^{^(})+2\right)\left((^{^(})-3\right)=0$$ Since square roots are always positive, we get that $(^{^(})=3$.

$x$ is a real number such that $x\in\left[0,\frac{\pi}{2}\right]$. If $\log_{24\sin x}(24\cos x)=\frac32$, calculate $\cot^2x$

Use change of base to get $$\log_{24\sin x}(24\cos x) = \frac{\log (24\cos x)}{\log (24\sin x)} = \frac{\log(24)+\log(\cos x)}{\log(24)+\log(\sin x)} = \frac 32.$$ Simplifying, $$2\log(\cos x)-3\log(\sin x) = \log(24)\implies \frac{\cos^2 x}{\sin^3 x}=24.$$Thus, we have $\cot^2 x=24\sin x.$ Let $\sin x=(^{^(})$. To solve for $(^{^(})$, note that $$\frac{1-(^{^(})^2}{(^{^(})^3} = 24\implies 24(^{^(})^3 +(^{^(})^2-1=0.$$One can easily see that $(^{^(})=\frac 13$ is the only solution that works, as the other solutions are imaginary, which is absurd. Thus, the answer is $24\cdot \tfrac 13 = \boxed{8}$.
